Overview

DjinnJS is an ES Module framework designed to assist developers in building performance-focused websites and web applications. Check out the project on GitHub or NPM.

Out of the box DjinnJS converts traditional websites into a Single Page Application (SPA) by implementing Pjax Navigation alongside an offline first content strategy. DjinnJS messaging can also used by UI Frameworks when building a traditional SPA.

What Is DjinnJS

  • Just In Time Resouce Fetching
  • Offline First Content Strategy
  • Pjax Navigation
  • ES Module Imports
  • Persistent State with Server Side Rendering
  • Context-Specific Progressive Enhancements
  • An Actor Model based Messaging System

Supported Environments

  • Modern browsers
  • Electron
  • Edge (IE 12 with polyfills)
